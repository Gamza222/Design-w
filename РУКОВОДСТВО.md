## 🚀 КАК ОТКРЫТЬ САЙТ

### Вариант 1 — Через браузер напрямую
Дважды кликни на файл `index.html` → откроется в Safari.

> ⚠️ Формы (отправка заявок) работают только через сервер.

### Вариант 2 — Через локальный сервер (рекомендуется)
Скажи Claude: **«запусти сервер»** — он сделает всё сам.
Затем открой в Safari: **http://localhost:8080**

После каждого изменения жми **Cmd+Shift+R** в Safari.

## 🎨 ДИЗАЙН (обновлён май 2026)

Сайт использует **3 основных цвета**:
- **Тёмно-синий** `#1A1A2E` — основной фон
- **Тёмное золото** `#C9A050` — акценты, кнопки, подчёркивания
- **Светлый нейтральный** `#E8E4DC` — текст на тёмном фоне

### Шапка сайта
Шапка прозрачная поверх главного фото. При прокрутке становится тёмной.

Структура шапки (одна строка):
```
[Лого] ——— [Дизайн-проект / О нас / Портфолио / Достижения / Блог / Контакты] / [TG] [VK] [WA] / [Телефон] / [10:00–19:00] / [Рассчитать проект]
```

Hover-эффект на ссылках: золотая полоска появляется под ссылкой слева направо.

## ✏️ ЧТО МОЖНО МЕНЯТЬ САМОСТОЯТЕЛЬНО

### Тексты
Открой нужный файл и меняй текст внутри тегов (между `>` и `<`).
Не трогай сами теги HTML и атрибуты `class=`, `id=`.

**Безопасно менять:**
- Тексты
- Номера телефонов
- Email-адреса
- Цены

### Фотографии
Замени файл с тем же именем:
- `hero-bg.jpg` — главное фото в шапке
- `gramota1.jpg` / `gramota2.jpg` — фото грамот
- `review1.jpg`...`review17.jpg` — фото отзывов

Нет файлов `blog1.jpg`, `blog2.jpg`, `blog3.jpg`.
Добавь фото с такими именами — блог заработает.

## 🤝 КАК РАБОТАТЬ С CLAUDE

### Что Claude делает сам:
- Меняет тексты, цвета, размеры
- Исправляет баги
- Добавляет новые секции
- Переделывает дизайн
- Запускает сервер

### Примеры запросов:

| Что хочешь | Как сказать |
|------------|-------------|
| Поменять цвет кнопок | «сделай кнопки синими» |
| Добавить секцию | «добавь блок с ценами» |
| Исправить баг | «ссылка не работает» |
| Поменять шрифт | «используй шрифт Montserrat» |

### Перед важными правками:
> «Сначала сделай резервную копию»

Если что-то пошло не так:
> «Верни всё обратно»
